# bioinformatics-tool
# 🧬 Beast Mode Bioinformatics Suite

**A comprehensive web-based bioinformatics tool for DNA/RNA analysis, protein translation, mutation detection, and sequence alignment.**

---

## 🚀 Overview

This is a full-featured bioinformatics web application built with **Python**, **Streamlit**, and **Biopython**. Designed for researchers, students, and biology enthusiasts, this tool provides real-time sequence analysis without requiring any local installation or complex setup.

**Built by a 12-year-old self-taught programmer passionate about bioinformatics and computational biology.**

---

## ✨ Features

### 🔬 Core Analysis Tools
- **DNA ↔ RNA Conversion**: Instant transcription and reverse transcription with length tracking
- **RNA → Protein Translation**: Complete amino acid sequence generation with composition analysis
- **Mutation Detection**: Advanced point mutation identification (Transitions & Transversions) with position mapping
- **Codon Usage Analysis**: Frequency distribution of all 64 codons in a given sequence

### 🧬 Advanced Bioinformatics
- **Pairwise Sequence Alignment**: Global alignment using Needleman-Wunsch algorithm with customizable scoring matrices
- **Phylogenetic Distance Matrix**: P-distance calculation for multiple sequences (foundation for evolutionary tree building)

### 📄 File Processing
- **FASTA Parser**: Extract and analyze sequences from standard FASTA files
- **FASTQ Parser**: Quality control analysis with PHRED score evaluation (Q10/Q20/Q30 thresholds)

- **Research**: Quick sequence analysis for small-scale bioinformatics projects
